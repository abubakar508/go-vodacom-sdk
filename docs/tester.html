Local C2B Tester

The real tester must run locally because API keys should not be entered into a public GitHub Pages page and browsers cannot reliably set the M-Pesa Origin header manually.

Run this from your cloned repo:

go run ./examples/c2b_tester

Then open:

http://localhost:8088
  • Origin header defaults to *.
  • You can enter your API key and platform public key.
  • It generates a SessionID and performs only C2B Single Stage.
  • Credentials are not stored by the tester.
